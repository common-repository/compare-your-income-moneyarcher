== Description ==

This plugin will allow to add 5 famous people and let user check their income compared to the famous people.
From V2.0.0 there are 15 options to compare and show the calculator.
One Page/Post will support one shortcode.
More features are getting added.

Plugin Features

* Easy configuration
* Customizable
* Responsive
* Lightweight
* Modern design

== Installation ==

* Upload the zip file to the `/wp-content/plugins/` directory and extract it.
* Activate the plugin through the 'Plugins' menu in WordPress.
* Now customize it as per your need.

== Frequently asked questions ==

= Will it slow down my website =

The answer is no. It is very light weight plugin.

= Does it incude JavaScript =

Yes! And it add JavaScript into the footer so it will not slow the page loading.
